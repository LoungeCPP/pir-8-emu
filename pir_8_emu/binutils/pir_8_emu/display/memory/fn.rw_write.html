pub fn rw_write(x_start: usize, y_start: usize)

Prepare the "Memory access" window at the specified coords

The window is 13x6, laid out as follows:

Memory access
<mem_acts>

Where <mem_acts> is either {none}, or up to 5 lines in the following format:

0000 ← 0x10
0001 → 0x11
0002 ≡ 0x12

Where memory at 0x0000 was written to and has value 0x10, memory at 0x0001 was read from and has value 0x11, and memory at 0x0002 was read from and written to and has value 0x12,
